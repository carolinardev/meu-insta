# 📸 Meu Insta

Um mini projeto inspirado no Instagram, desenvolvido com HTML, CSS e JavaScript 💖

## ✨ Funcionalidades

- 📝 Criar publicações
- ❤️ Curtir posts
- 💬 Comentar
- 🗑️ Excluir publicações
- 📷 Upload de imagens
- 🌙 Modo claro e escuro
- 📱 Layout responsivo (mobile)

## 🎨 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## 🚀 Como usar

1. Baixe o projeto
2. Abra o arquivo `index.html`
3. Pronto! 🎉
